# Binary Heap / Priority Queue in C#

This project implements a generic Binary Heap (Priority Queue) in C# using .NET. The project demonstrates core object-oriented programming concepts, generics, custom comparers, and heap-based algorithms.

## Features

- Generic heap implementation
- Min-heap behaviour using custom comparers
- Insert and remove operations
- Heapify functionality
- Priority queue behaviour
- Console-based testing

## Technologies

- C#
- .NET
- Object-Oriented Programming (OOP)
- Generics
- Interfaces
- Data Structures & Algorithms

## Project Structure

- `Heap.cs` – Core heap implementation
- `IHeapifyable.cs` – Heap interface definition
- `Tester.cs` – Test cases and validation
- `.csproj` – .NET project configuration

## Concepts Demonstrated

This project demonstrates:

- Encapsulation
- Abstraction
- Interfaces
- Generic programming
- Algorithm implementation
- Problem solving
- Testing and debugging
